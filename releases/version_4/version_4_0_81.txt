==============
版本 4.0.81
==============

Jam.py:

* 修正了无模式窗体中字段的显示问题
* 为客户端应用方法添加了异步参数
* 修复了当展开的属性为 false 时克隆方法出现的缺陷
* 修复了与字段及参数的 lookup_value 、lookup_text 和 display_text 属性相关的漏洞
* 修复了用户角色被禁用 “可查看(Can view)” 权限时，仍可打印报告的问题
* 服务器端的 open、set_where、set_fields、set_order_by 方法，
  可使用与客户端对应方法相同的参数
* 修改了客户端的 edit_record、apply changes、cancel_edit 方法，
  使其在 can_edit 方法返回 false 时，用户仍能打开文档查看
* 转换报表时，不向系统传递还原参数
* 移除了 index.html 文件中 CSS 和 JS 链接开头的 “/” 符号


管理员:

* 更正了字段名称的验证


演示应用程序:

* 选择添加到发票编辑表单中的按钮，将所选曲目添加到发票中
* set_view 方法返回 false 的可见项不会添加到动态菜单中
